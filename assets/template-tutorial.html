品牌配色方案

配色是设计系统的灵魂。花心小泡泡的配色方案围绕暖棕色系展开， 以 #A67B5B 为主色调，辅以柔和的黄色 #F0C674 和克制的蓝色 #5B8BA0， 整体呈现出温暖、自然、专业的视觉感受。

在具体应用中，棕色用于主要行动按钮和强调元素，黄色用于高亮和点缀， 蓝色用于链接和信息提示。背景采用奶油色 #FDF8F0 营造舒适的阅读环境。

字体排版系统

我们选择了 Fraunces 作为展示字体，搭配 Noto Sans SC 作为正文字体。 Fraunces 优雅的衬线特征赋予标题独特的品牌辨识度， 而 Noto Sans SC 的清晰易读保证了长篇内容的阅读体验。

代码块使用 Fira Code 等宽字体，手写体使用 Caveat 增加亲切感。 通过 clamp() 函数实现流式字体大小，确保在任何屏幕尺寸下都能获得最佳阅读体验。

间距与布局

采用 4px 基准的间距体系，所有间距值都是 4 的倍数。 页面最大内容宽度通过 clamp(320px, 90vw, 1100px) 动态控制， 在保证可读性的同时充分利用屏幕空间。

布局策略上采用不对称双栏、粘性侧边栏、三等分网格等多种模式， 根据内容类型灵活切换，避免单调的单一布局。

组件设计规范

所有组件都遵循统一的设计语言：圆角 8-16px、柔和的阴影、 微妙的边框、以及一致的交互反馈。卡片组件使用白色背景浮于暖色底上， 悬停时微微上浮并增强阴影，营造层次感。

动效与过渡

页面滚动时使用 IntersectionObserver 触发元素渐入动画。 过渡时长统一为 0.3s，使用 ease-out 缓动函数， 确保动画流畅自然而不突兀。

可访问性

所有文本与背景的对比度都满足 WCAG AA 标准。 交互元素具有清晰的焦点样式，支持键盘导航。 使用语义化 HTML 标签确保屏幕阅读器友好。

构建流程

从零开始搭建设计系统

1

定义品牌基因

梳理品牌核心价值，提取关键视觉元素，确定主色调、辅助色、字体组合和设计原则。

2

建立设计令牌

将配色、字体、间距、圆角等基础属性抽象为 CSS 变量，形成可复用的设计令牌体系。

3

构建组件库

基于设计令牌开发按钮、卡片、导航栏等基础组件，确保每个组件都有完整的交互状态。

4

编写页面模板

组合组件形成页面模板，覆盖常见业务场景，为新项目提供快速启动的基础。

5

持续迭代优化

收集使用反馈，持续完善设计系统，保持与业务需求同步演进。
