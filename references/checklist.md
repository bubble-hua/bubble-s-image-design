# Checklist / 自检清单

> 所有页面交付前必须逐条检查。P0 必须全过，P1 应过，P2 加分。

---

## P0 — 必须全过（任何一条不过就要改）

### 配色

- [ ] **品牌三色**：棕 `#A67B5B` / 黄 `#F0C674` / 蓝 `#5B8BA0` 正确使用
- [ ] **三色比例**：主色60% + 强调色30% + 点缀色10%（点缀色永远是点缀）
- [ ] **背景色**：暖底 `#FDF8F0` 或交替 `#F5EDE0`
- [ ] **文字色**：墨色 `#2D2420`（正文）、浅墨 `#5C4D44`（次要）
- [ ] **无纯黑 `#000` 或纯白 `#fff`**
- [ ] **无AI常见配色**：蓝紫渐变、cyan、neon、冷灰蓝调

### 深色版面

- [ ] 3:4图文卡片场景：**绝对禁止深色底**
- [ ] HTML全屏页面：暗色面板 `#1E1A16` 可用

### 字体

- [ ] 标题用衬线体（汇文明朝体 / Noto Serif SC）
- [ ] 正文用无衬线体（Noto Sans SC + 系统栈）
- [ ] 英文装饰用 Fraunces italic
- [ ] 手写注释用 Caveat
- [ ] 代码/终端用 Fira Code
- [ ] **无 Inter / Roboto / Arial** 等overused字体

### 布局

- [ ] 每个Section布局不同（不能千篇一律）
- [ ] 内容宽度约束：`max-width: 1300px` + `margin: 0 auto`
- [ ] 间距使用 `clamp()` 做fluid sizing
- [ ] 响应式断点：900px / 600px
- [ ] **不是所有section居中**

### 组件

- [ ] **无浏览器默认样式**：无默认 `<blockquote>`、无默认 `border-left` 引用、无默认 `<ul>/<ol>`、无默认 `<table>`
- [ ] 所有组件从 `components.md` 选用或自行设计符合 brand-dna 的组件
- [ ] **强调色装饰边框不低于40px**（统一40px）

### 品牌

- [ ] 署名：花心小泡泡 / Bubble
- [ ] 签名档：花心小泡泡 · 体验生活，感受体验，记录感受
- [ ] CTA文字：Follow 在下 花心泡

---

## P1 — 应过（尽量满足）

### 视觉

- [ ] 字号对比极端（Hero大 + 辅助小）
- [ ] 有至少一种材质/肌理感（条纹、网格、彩铅、水彩）
- [ ] 有至少一种装饰元素（虚线圆圈、渐变光晕、渐隐线、大透明数字）
- [ ] 选中文本高亮：`::selection { background: #F0C674; color: #2D2420; }`
- [ ] 链接悬停有反馈（底色块或下划线）

### 布局

- [ ] Section间距：`clamp(80px, 12vh, 160px)`
- [ ] 内容块间距：`clamp(40px, 6vw, 100px)`
- [ ] 卡片内padding：`clamp(28px, 3vw, 44px)`
- [ ] 响应式是"重新排列"不是"缩小"
- [ ] 移动端不隐藏内容

### 动效

- [ ] 无 bounce / elastic 动效
- [ ] 无 animate width/height
- [ ] 无无限循环动画
- [ ] 尊重 `prefers-reduced-motion`

---

## P2 — 加分（锦上添花）

### 品牌辨识度

- [ ] 一眼能认出是花心小泡泡的品牌
- [ ] 截图发社交媒体不会被说"又是AI做的"
- [ ] 有设计师眼光（细节讲究、间距精确、色彩克制）
- [ ] 有个人品牌感

### 旅行主题

- [ ] 有旅行主题装饰（机票/邮票/地图/拍立得/手账贴纸 至少1种）
- [ ] 有坐标标注或地图元素

### 材质感

- [ ] 有彩铅笔触或手绘感元素
- [ ] 有水彩晕染或渐变光斑
- [ ] 有纹理肌理（条纹/网格）

### 排版细节

- [ ] 有出血/溢出效果（图片超出容器边界）
- [ ] 有大透明数字做section装饰
- [ ] 有高亮标记（文字底部色块）
- [ ] 行间距/字间距肉眼检查无异常

## 🚫 通用禁忌清单（参考）

| 类型 | 禁止 |
|------|------|
| 配色 | 蓝紫渐变、cyan、neon、纯黑白、AI常用的冷灰蓝调、颜色过渡失调 |
| 深色版面 | 3:4图文卡片禁止黑色/深色版面；HTML全屏可用深色面板 |
| 字体 | Inter/Roboto/Arial等overused字体、monospace充当"技术感" |
| 布局 | 所有section居中、千篇一律卡片网格、cards嵌套cards |
| 动效 | bounce/elastic、animate width/height、无限循环动画 |
| 装饰 | glassmorphism、圆角矩形+阴影千篇一律、渐变文字、AI光效 |
| 整体 | 看起来像AI生成的通用模板、generic Landing Page模板感、小家子气 |
| 边框 | 强调色装饰边框细于40px |
| 排版 | 行间距/字间距异常 |
| 图片 | AI生成的stock photo风、过度滤镜、无意义装饰图 |
| 默认样式 | HTML默认blockquote、默认border-left引用、无样式ul/ol、默认table |

## 🔍 自检问题

做完设计后必须问自己：

1. **这个页面截图发到社交媒体，会不会被人评论"又是AI做的"？**
2. **能不能一眼认出这是花心小泡泡的品牌？**
3. **有没有哪个部分让你觉得"见过很多次了"？**

三个问题的答案应该是：不会 / 能 / 没有。
